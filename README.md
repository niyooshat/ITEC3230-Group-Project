# Veridian - Plant Identification and Care Platform

## Project Description
Veridian is a web-based platform that is designed to help users identify plants, manage their gardens, track plant care activities, and engage within a community.

## Features
- User authentication (sign in / register)
- Plant search and filtering
- Plant identification
- Plant details with care information
- Personal garden management
- Care activity logging
- User profile management
- Community posts and gallery
- Create, edit, and view posts
- Commenting system

## Technologies USed
- HTML5
- CSS3
- JavaScript
