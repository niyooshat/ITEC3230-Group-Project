# Veridian - Plant Identification and Care Platform

## Project Description
Veridian is a web-based platform that is designed to help users identify plants, manage their gardens, track plant care activities, and engage within a community.

## Features
- User authentication (sign in / register)
- Plant search and filtering
- Plant identification
- Plant details with care information
- Personal garden management
- Care activity logging
- User profile management
- Community posts and gallery
- Create, edit, and view posts
- Commenting system

## Technologies USed
- HTML5
- CSS3
- JavaScript

## Implementation Approach
The platform has been implemented as a multi-page front-end prototype using HTML, CSS, and JavaScript.

Each major feature is separated into its own HTML page to clearly represent different parts of the system.

A single CSS file is used to maintain a consistent style across all of the pages. This includes the layout structure, colour scheme, typography, buttons, and reusable components such as cards and modals.

JavaScript is used to handle interactive elements such as:
- opening and closing modal windows (login, identify plant, make/edit post)
- handling the button interactions

This design follows a consistent layout with a shared navigation bar across all pages to support usability and easy navigation.
